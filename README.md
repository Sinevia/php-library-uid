## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-uid.git"
        }
    ],
    "require": {
        "sinevia/php-library-uid": "dev-master"
    },
```
